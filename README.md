# Spotify UI Clone (HTML + CSS)

Spotify-inspired music player UI built with pure HTML and CSS for layout practice.

## Features
- Sidebar navigation, library, and call-to-action cards
- Sticky top navigation bar
- Content sections with a responsive card grid
- Fixed bottom music player with controls and sliders

## Tech Stack
- HTML5
- CSS3 (Flexbox, sticky positioning)
- Font Awesome (icons)

## Getting Started
1. Clone the repository.
2. Open `index.html` in your browser.

No build tools or JavaScript required.

## Project Structure
- `index.html` — markup
- `style.css` — styles
- `assets/` — images and icons
